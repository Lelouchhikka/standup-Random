# Настройка JSONBin для хранения данных

## Шаг 1: Создайте аккаунт на JSONBin.io

1. Перейдите на https://jsonbin.io
2. Нажмите "Sign Up" и зарегистрируйтесь (можно через GitHub)
3. Подтвердите email

## Шаг 2: Создайте новый Bin

1. После входа нажмите "Create Bin"
2. Вставьте следующий JSON в редактор:

```json
{
  "participants": [],
  "lastUpdated": null,
  "totalParticipants": 0,
  "updatedBy": "Initial setup"
}
```

3. Нажмите "Create"
4. Скопируйте **Bin ID** из URL (например: `65f8a1b2c9e7e8001234abcd`)

## Шаг 3: Получите API Key

1. В правом верхнем углу нажмите на свой профиль
2. Перейдите в "API Keys"
3. Скопируйте **Master Key** (начинается с `$2a$`)

## Шаг 4: Добавьте переменные окружения в Vercel

1. Откройте ваш проект в Vercel: https://vercel.com/dashboard
2. Выберите проект "standup-Random"
3. Перейдите в **Settings** → **Environment Variables**
4. Добавьте две переменные:

### JSONBIN_BIN_ID
- **Name**: `JSONBIN_BIN_ID`
- **Value**: ваш Bin ID (например: `65f8a1b2c9e7e8001234abcd`)
- **Environment**: Production, Preview, Development

### JSONBIN_API_KEY
- **Name**: `JSONBIN_API_KEY`
- **Value**: ваш Master Key (например: `$2a$10$...`)
- **Environment**: Production, Preview, Development

5. Нажмите "Save" для каждой переменной

## Шаг 5: Передеплойте проект

После добавления переменных окружения:

1. Перейдите на вкладку **Deployments**
2. Нажмите на три точки у последнего деплоя
3. Выберите "Redeploy"
4. Подтвердите действие
